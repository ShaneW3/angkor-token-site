About the Project / អំពីគម្រោង

Angkor Token is a blockchain-based initiative to support Angkor Wat. We collaborate with Cambodia’s tourism authorities. Tokens help fund site restoration and enhance tourist engagement.
Angkor Token គឺជា​គម្រោងប្រើប្រាស់ប្លុកខែនដើម្បីគាំទ្រវត្តអង្គរវត្ត។ យើងមានការសហការជាមួយអាជ្ញាធរទេសថាផលសកម្មភាព។ Token ក្រោមឧបត្ថម្ភដើម្បីស្ថាបនាគម្រោងបង្រ្កាបបំពាននិងបម្រើការសកម្មភាពនៃទេសចរ។
